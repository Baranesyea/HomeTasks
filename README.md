# משימות הבית 🏠

לוח משימות יומי ל-4 ילדים — שיה, מיכאל, יער ודין.
עמוד יחיד, ללא שרת, ללא לוגין. רץ מ-GitHub Pages.

## איך זה עובד
- כל בוקר ב-06:00 הרשימה מתאפסת אוטומטית.
- משימות "מסתובבות" מתחלפות בין הילדים לפי תאריך (דטרמיניסטי, הוגן).
- כל סימון נשמר ב-localStorage עד הבוקר הבא.
- טור של ילד הופך ירוק כשסיים את כל המשימות שלו.
- סקשן שלם הופך ירוק כשכולם סיימו, וכשכל היום ירוק יש קונפטי 🎉

## מצב הורה
טאפ 5 פעמים על התאריך → נפתח עורך JSON של רשימת המשימות.

## טכנולוגיה
HTML + Tailwind (CDN) + Vanilla JS. PWA עם Service Worker.
